    • Categories: Software

    Es gibt einen neuen Stern am Softwarehimmel und das ist die Wahlkampf-App der LINKEN. Als App für das Tablet oder Smartphone aus den App Stores installierbar kann sie auch als Webseite auf dem heimischen PC geöffnet werden. Gebaut wurde sie als Progressive Web App. Es können Events auf einer Karte angelegt werden und Leute zu den Events eingeladen werden.
